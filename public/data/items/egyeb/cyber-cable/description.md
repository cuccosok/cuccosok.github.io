# Cyber Coiled Cable

## Prémium USB-C Spirálkábel

Kézzel készített, **aviator csatlakozós** spirálkábel amely nem csak funkcionális, de stílusos kiegészítője az asztalodnak.

### Jellemzők
- **Aviator** gyorscsatlakozó
- **Double-sleeved** paracord borítás
- **USB-C** mindkét végén
- Választható színek: Cyan / Pink / Dual

*Minden kábel egyedi, kézzel tekert.*
