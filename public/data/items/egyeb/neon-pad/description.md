# Neon Deskpad XL

## Az Asztalod Új Arca

A **Neon Deskpad XL** a teljes asztalt beborító, cyberpunk mintás egérpad RGB peremvilágítással.

- **RGB** peremvilágítás 12 zónával
- **Micro-textured** felület a tökéletes kontrollért
- **Csúszásgátló** gumi alap
- **Vízálló** felső réteg
