# Neon TKL Pro

## A végső gamer billentyűzet

A **Neon TKL Pro** a versenyszerű játékosok számára készült, villámgyors válaszidővel és megbízható kapcsolókkal.

### Jellemzők
- **1ms** válaszidő vezetékes módban
- **N-key rollover** — minden gombnyomás regisztrálva
- **Kailh Box** kapcsolók IP56 por- és vízvédelemmel
- **Mágneses** csuklótámasz mellékelve
- **Macro** programozás dedikált szoftverrel

### Specifikációk
| Tulajdonság | Érték |
|---|---|
| Layout | TKL (87 gomb) |
| Polling rate | 8000Hz |
| Kábel | Levehető USB-C fonott |

*Limitált cyberpunk kiadás — csak 500 db készül.*
