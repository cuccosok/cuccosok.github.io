# Phantom X1

## Ultralight Gaming Egér

A **Phantom X1** mindössze 49 grammos, mégis rendkívül strapabíró és precíz gaming egér.

### Jellemzők
- **PAW3395** szenzor — 26000 DPI
- **49g** ultrakönnyű honeycomb dizájn
- **Kailh 8.0** optikai kapcsolók (100M kattintás)
- **Paracord** kábel / Wireless dual-mode
- **PTFE** csúszótalpak

*Az éjszaka szelleme — láthatatlan, de halálos.*
